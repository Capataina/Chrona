## Conventions

- **Filename format:** `NNN_<short_topic>.md` (NNN is zero-padded sequence number)
- **Status values:** `planned`, `in_progress`, `blocked`, `complete`
- **Checklists:** `[ ]` pending, `[x]` done
- **Single source of truth:** All implementation state, decisions, and next steps live here
